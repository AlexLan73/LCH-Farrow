Нужно создать новый генератор сигналов на GPU
Нужно взять настройки из и влгоритм из LFMSignalGenerator

он так же будет создавать ЛЧМ сигнал только на GPU
класс типа GenegstorGPU

структура проекта примерно такая
- по возможности все функции делаем на GPU на OpenCl
src
  generator
     GenegstorGPU - генерируем сигналы в GPU и передаем адреса для дальней шего использования
      - signal_base - Формирует Базовый сигнал ЛЧМ из параметров (базовые точки для расчета дробной задержки), записываем в GPU и возвращаем ссылку на память. 
      - signal_valedation - сформировать аналитическим (расчетным) путем вектора с задержкой по тем же данным что передаются в  FuncGPU::fractional_delay
            , записываем в GPU и возвращаем ссылку на память.

  gpu
     FuncGPU  - класс реализует набор функций на GPU  
      - fractional_delay - функция реализует дробную задержку к переданному сигналу 
          - параметры входные:
              адрес входного сигнала на GPU
              массив номер луча задержка в градусах (шаг 0.5 градуса) m_delay[index]=(номер луча, задержка)
          - параметры выходные:
              Возвращает ссылку на вектор с новыми значениями по лучам 
              ray_all = [ray0 -> значения после сдвига][ray1 -> значения после сдвига].. [rayn-1 -> значения после сдвига]
      - ClearGPU() - очищает память на GPU

      - func_valedation - получает две ссылки на данные на GPU из GenegstorGPU::signal_valedation -базовые значения для валидации и ractional_delay - значения после задержки
          - результат сравнить эти сигналы по лучам и найти отличие с заданной точностью (если у нас дробная задержка реализована Лагранжем 48*5 )
